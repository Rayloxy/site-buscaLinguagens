Python

Python é uma linguagem de programação de alto nível, interpretada, de script, imperativa, orientada a objetos, funcional, de tipagem dinâmica e forte. Categoria: Web, Data Science, AI.

Java

Java é uma linguagem de programação orientada a objetos, baseada em classes e concorrente. É conhecida por sua portabilidade com o lema "write once, run anywhere". Categoria: Backend, Mobile (Android), Enterprise.

JavaScript

JavaScript é uma linguagem de programação interpretada de alto nível, caracterizada como dinâmica, fracamente tipada, baseada em protótipos e multiparadigma. Categoria: Web, Frontend, Backend.

Lua

Lua é uma linguagem de programação leve, de script e multiparadigma, projetada para ser facilmente embarcada em outras aplicações para estender suas funcionalidades. Categoria: Games, Scripting, Embedded Systems.

C#

C# (pronuncia-se "C sharp") é uma linguagem de programação moderna, orientada a objetos e de tipo seguro, desenvolvida pela Microsoft como parte da plataforma .NET. Categoria: Backend, Games (Unity), Windows.
